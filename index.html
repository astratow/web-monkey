Glass Supplies

Website made for Glass Supplies , double glazing window units manufacturer.

This is our largest project so far. We did it using basic and free tools to minimize clients cost, yet provide responsive, functional and modern website.

We provided Search Engine Optimization, which driven website up to top 10 search results. We also provided SSL certification, redirected domain to new hosting (old hosting was expensive and not very helpful).

  • Category: Web Development
